What is PyScout?

A focused utility for finding the live switchport.

PyScout helps network engineers, sysadmins, desktop support, and field techs answer the access-layer question quickly: which switchport is this network drop connected to?

Built for practical field work

Use it while tracing wall jacks, checking patch-panel records, troubleshooting user drops, and collecting switchport-to-location notes.

Features

Purpose-built access-layer tools.

PyScout keeps the output centered on the information technicians need at the drop.

SW

Switchport discovery

Identify the connected switch and interface before opening a switch session.

L/C

LLDP/CDP support

Use common discovery advertisements from the connected access switchport.

MAP

Physical mapping

Capture switchport-to-location notes for wall jacks, patch panels, and field records.

LAN

Local subnet scan

Check a local subnet for basic device visibility during hands-on troubleshooting.

OUT

Clean output

Show useful results without raw dumps, buried fields, or manual parsing.

EXE

Windows executable

Run PyScout on Windows workstations used by support teams and field technicians.

How to use

From network drop to documented port.

The workflow is intentionally short for troubleshooting on a laptop in the field.

  1. 1

    Plug into network drop

    Connect to the wall jack or patch point you need to identify.

  2. 2

    Run pyscout.exe

    Launch the Windows executable from the release package.

  3. 3

    Start discovery

    Read the connected switch, port, protocol, and IP from the result view.

  4. 4

    Save mapping notes

    Record the physical location so the next technician has a clean handoff.

Requirements

What PyScout needs.

PyScout depends on discovery advertisements from the connected access switch.

Windows

Built for technicians working from Windows laptops and workstations.

Npcap

Required for local discovery capture on the connected interface.

LLDP or CDP enabled on the switchport

The connected access switch must advertise discovery information.
