Network field tool

Know the switchport before you leave the closet.

Py-Scout is a Windows desktop tool for discovering the switch, port, neighbor address, and protocol seen from the jack you are plugged into.

Built for network engineers doing physical mapping, closet cleanup, drop validation, and field troubleshooting.

Workflow

Plug in. Discover. Save the mapping.

01

Connect

Plug your laptop into the wall jack, patch cable, or switchport being traced.

02

Run discovery

Py-Scout listens for neighbor discovery frames and normalizes the useful fields.

03

Document

Save the result with location, wall jack, patch panel, or field notes.

Why it exists

A practical mapping tool, not a packet viewer.

Py-Scout keeps the field workflow simple: identify the upstream switchport, capture the result, and build a clean record you can use later.

Switchport discovery

Find the advertised switch, interface, management address, and discovery protocol.

Readable results

Shows the fields an engineer needs without forcing you to inspect raw frames.

Persistent mapping

Keep a local table of known ports, locations, and notes for repeatable documentation.

Windows-first GUI

Designed around a desktop workflow instead of a pile of one-off terminal commands.

Field-friendly output

Useful during closet work, jack validation, refresh projects, and troubleshooting calls.

Engineer-owned

Small, inspectable, and built for real network operations instead of generic inventory.

Mapping

Built for the part most tools ignore.

Discovery is only step one. The real value is turning a live switchport result into a persistent physical map.

  • Switch and interface
  • Protocol seen from the drop
  • Neighbor management address
  • Wall jack or room location
  • Patch panel reference
  • Notes for cleanup or follow-up work

Install

Requirements

  • Windows
  • Npcap
  • Network interface connected to the target drop
  • LLDP or CDP enabled on the upstream switch
