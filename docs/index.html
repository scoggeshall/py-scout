What Py-Scout does

Answers the access-layer question first.

  • Identifies switch and port using LLDP/CDP
  • Shows clean, readable discovery output
  • Saves physical mappings
  • Provides a quick subnet scanner for local visibility

Use it when

Field work needs a clear switchport answer.

  • Tracing unknown wall jacks
  • Verifying patch panel mappings
  • Troubleshooting user drops
  • Auditing switchport usage
  • Cleaning up access-layer documentation

Example output

Readable output for the thing you need to know.

Switch: MDF-SW01
Port: Gi1/0/24
IP: 10.16.1.45
Protocol: CDP

Built for access-layer work

Requirements

What you need

  • Windows
  • Npcap
  • LLDP/CDP enabled on the switchport
