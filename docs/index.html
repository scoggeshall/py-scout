Switchport discovery

Find your switch port instantly

Plug in, run Py-Scout, and immediately see the switch and port using LLDP/CDP.

No tshark. No guesswork. Just the answer.

Download latest release View source

Workflow

How it works

  • Plug into a network jack
  • Run discovery
  • See switch + port instantly
  • Save mapping if needed

Value

What you get

  • LLDP + CDP switchport discovery
  • Readable output (no packet dumps)
  • Single Windows executable
  • Table-based mapping system
  • Manual or Auto Save

Result

Example output

Switch: MDF-SW01
Port: Gi1/0/24
IP: 10.16.1.45
Protocol: CDP

Discovery

Protocols

Py-Scout supports:

  • LLDP
  • CDP
