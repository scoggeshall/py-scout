Py-Scout

A GUI-first switchport discovery and physical mapping tool for network engineers.

Plug in, run Scapy-powered LLDP/CDP discovery, see the connected switch and port, then save the physical mapping to SQLite.

Core Workflow

Plug in. Discover. Map. Done.

Py-Scout is intentionally focused on one field workflow: identify the switchport a machine is connected to and preserve the physical mapping record.

Run Discovery

Select or auto-detect a capture adapter and listen for LLDP/CDP neighbor advertisements with Scapy.

Review Results

See the switch name, switch port, neighbor IP, protocol, and discovery timestamp in a clean desktop interface.

Save Mapping

Store site, building, room, wall jack, patch panel, switch, port, host details, and notes in SQLite.

Active Product

Focused on discovery and physical mapping.

  • Modern PySide6 desktop GUI with Discovery and Mapper tabs.
  • Scapy-based LLDP/CDP discovery and adapter auto-selection.
  • SQLite-backed mapper records with table-first editing and delete.
  • Single windowed Windows executable: dist\pyscout.exe.

Supported Discovery Protocols

LLDP and CDP are both supported.

Py-Scout owns the capture and parsing pipeline directly with Scapy, then normalizes the result into switch, port, neighbor IP, protocol, timestamp, and backend fields.

The Discovery tab can listen for both protocols, LLDP only, or CDP only.

LLDP

Parses standard LLDP TLVs for system name, port identity or description, and IPv4 management address.

CDP

Detects Cisco LLC/SNAP CDP frames and parses device ID, port ID, IPv4 address, capabilities, software version, and platform.

Packet Capture Requirement

Windows capture requires Npcap.

Scapy handles discovery inside Py-Scout, but Windows still needs Npcap or compatible packet capture support installed for live captures.

Clean Errors

Missing packet capture support is reported as a clear Npcap requirement instead of a Python traceback.

Requirements

Built for Windows field use.

  • Windows
  • Npcap or compatible packet capture support
  • Python 3.10 or newer for source development
  • PySide6 for source GUI runs
  • PyInstaller for executable builds

Discovery depends on LLDP or CDP advertisements reaching the local adapter. Some docks, phones, unmanaged switches, or filtered links may block discovery traffic.

Build

Create the GUI executable from source.

The packaged app is GUI-first. Source CLI support is intentionally minimal and kept for development checks only. 

python -m unittest discover
python -m compileall pyscout
python -m pyscout --help
python -m pyscout version
.\build.ps1
.\dist\pyscout.exe
